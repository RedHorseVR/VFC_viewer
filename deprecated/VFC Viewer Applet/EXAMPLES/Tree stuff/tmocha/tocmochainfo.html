TOCmocha

TOCmocha is a simple and compact Table of Contents JavaTM applet. One-level folders and frames-based navigation with option to specify the target-frame for each content line are supported. The applet loads a pair of images for each content line and is based on JDK 1.0.2. A demo of this applet to navigate this site is at the left frame of this page.

The applet generates the TOC based on the data provided in a simple text file.

Usage

  • If you are a Netscape User please read Instructions for Netscape Users.
  • Single-click on the folder to open/close.
  • Single-click on any folder item to load the page in the specified frame. The specification of which frame to load the page for a given content line is defined in tocmocha.txt.
  • A demo of how this applet can be used to support multiple sub-folders is similar to the one described in the text version of this applet, instantTOC.

Download

This applet does not have any limit to the number of content lines. You are welcome to use TOCmocha at your site. Here are the various files:
tocmocha.txt, tocmocha.class, tocmochaLine.class, tocmochainfo.html, tocmochahome.html and tocmocha.html.

If you are using Netscape Communicator 4.05 to test the applet locally and if you have downloaded the applet to a directory C:\test then you need to specify codebase="file:///C:/test/" and specify the value of TXTFILE to be "file:///C:/test/tocmocha.txt".
